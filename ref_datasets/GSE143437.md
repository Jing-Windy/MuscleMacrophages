## GSE143437
# preprocessing
- This dataset features a cluster of cells with high mitochondria gene expression (10 < pctMt < 20). It is also a prefiltered dataset (De Micheli et al., Cell Rep. 2020). QC bar was changed accordingly. minGene was set to 600 to remove remaining low quality cells.<br>
minGene = 600<br>
pctMT = 20<br>
maxGene = 5000<br>
maxCount = 20000<br>
minCount = 1000<br>

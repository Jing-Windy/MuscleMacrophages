# GSE195507
## preprocessing
- QC threshold<br>
Samples in GSE195507 all have relatively higher gene counts, so we loose restriction on maxCounts to avoid discarding too many cells.<br>
    > minGene = 500<br>
    > pctMT = 10<br>
    > maxGene = 8000<br>
    > maxCount = 30000<br>
    > minCount = 1000<br>

There are some other cells with extremely low counts and features, but we proceed with the current threshold first.<br>
![QC result](images/GSE195507/posQC1.png)<br>
It seems that those cells exist both sporatically and as a cluster. (low: nFeatureRNA < 1200)<br>
![postQC FeaturePlot](images/GSE195507/lowFeature_1200.png)<br>
- Remove doublets<br>
Doublets are detected by scDblFinder. Samples are processed separately. For this dataset each sample is loaded with a relatively small cell number, so expected doublet rates are also lower. Only tens of doublets are detected in each sample.
![Doublet FeaturePlot](iamges/GSE195507/scDblFinder.png)
- Integration and clustering
Little batch effect was observed. Currently with a resolution of 0.6 we have 14 clusters to discuss with.
![Harmony](images/GSE195507/Harmony.png)
![seurat_clusters](images/GSE195507/clusters_0.6.png)

## Main characteristics
- high UMI & high feature counts
- some cells with few feature genes
